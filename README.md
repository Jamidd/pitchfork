## Requerimientos:
* python 3.8
* node 10.9
* npm 6.13

## Instrucciones:
* 1) En la consola, ubicado en el directorio principal ejecutar los siguientes comandos.

```
> pip3 install -r requirements
> cd webpage
> npm install
> npm run
```

* 2) Entrar al browser con la URL: http://localhost:3000/
